---
title: Starship Configuration
---

Add this section to your `starship.toml` and get a nice info prompt about your slowfoot project.

```toml
[custom.slowfoot]
command = 'vendor/bin/slowfoot starship'
detect_files = ['slowfoot-config.php']
```
